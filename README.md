# excel-sales-performance-dashboard
Sales Performance Dashboard (Excel)
Project Overview

This project analyzes customer purchasing behaviour and revenue performance using an interactive Excel dashboard built with Power Query and Pivot Tables.

The goal was to transform raw transaction data into meaningful business insights and interactive visualizations.

## Tools Used

Microsoft Excel

Power Query

Pivot Tables

Pivot Charts

Interactive Slicers
## Dashboard Preview
## Dashboard Features

KPI cards for key metrics

Revenue analysis by product category

Seasonal revenue trends

Payment method analysis

Geographic revenue distribution

Customer purchase frequency analysis

Gender distribution insights

## Key Metrics

Total Revenue

Total Customers

Repeat Customers

Average Rating

Repeat Customer Percentage



## Key Insights

Clothing and accessories contribute the largest share of revenue.

Repeat customers account for the majority of transactions.

Digital payment methods dominate transaction preference.

Purchase frequency strongly influences revenue contribution.
